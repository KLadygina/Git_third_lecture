## Списки

Чтобы добавить не нумерованные списки, необходимо вначале поставить звездочку (*). 
Например вот так:
* Элемент1
* Элемент2
* Элемент3

Чтобы добавить нумерованные списки, необходимо просто пронумеровать тункты
Например вот так:
1. Первый пункт
2. Второй пункт
